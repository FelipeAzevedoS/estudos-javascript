# Estudos de JavaScript

Repositório de projetos simples e práticos para solidificar os fundamentos de JavaScript do zero.
Cada projeto foi construído sem tutoriais, apenas enunciado, documentação e tentativa e erro.

## Projetos

### Projeto 01 - Mudar texto ao clicar
Uma página que exibe frases de uma lista e as troca a cada clique num botão, voltando ao início ao chegar na última.

**Conceitos praticados:** manipulação do DOM, eventos de clique, arrays, operador módulo ('%'), 'textContent'.
